Fully Automatic Assembly Systems

Fully automatic assembly lines interconnected using a flexible circulation system for workpiece carriers or integrated collaborative robots.

System Parameters

  • Cycle time per battery pack: 30 seconds
  • Feeding of pre-assembled battery packs
  • Resistance welding of cell connectors to individual batteries
    • 0.8 seconds per weld with evaluation of welding parameters
    • Up to 60 welds per battery pack
  • Final assembly of the battery packs
  • EOLT of the battery packs
    • Charging current
    • Charging time
    • Functional test
  • Double-sided laser marking

System Parameters

  • Cycle time per power tap: 6.5 seconds
  • Feed and pre-assemble different sheet metal parts
  • Load and unload injection molding machine
  • Final assembly/final testing of the assembled power taps
  • Place components in trays

System Parameters

  • Cycle time per syringe: ≤ 0.5 seconds
  • Feed filled syringes, align them, and prepare them in sets of 10 for packaging
  • Simultaneously grip 10 syringes, rotate, and place them in pre-positioned blister packs in a defined orientation

System Parameters

  • Cycle time per control valve: 6.5 seconds
  • Feed coil bodies and assemble with contact pins
  • Feed copper wire and wind onto coil body
  • Feed, bend, and assemble wired components (diodes)
  • Resistance welding of contacts
  • Feed and assemble sheet metal parts
  • Laser welding (360° around) of sheet metal parts
  • Final inspection and preparation for plastic encapsulation
  • 4-fold plastic encapsulation of the component
  • Component testing and palletizing

System Parameters

  • Cycle time per connector: 5 seconds
  • Automatically feed and assemble various sheet metal parts
  • Package components into cartons
  • Seal cartons, label them, and stack them on euro pallets

System Parameters

  • Cycle time per connector strip: 7.5 seconds
  • Punch and pre-assemble different contact pins from carrier tape
    • Up to four different contact pins per connector possible
    • 42-pin connector and 30-pin connector (0.8 seconds per pin)
  • Load and unload injection molding machine (4 cavities)
  • Test overmolded connector strip
    • High voltage test
    • Wobble circle test of the pins
  • Place connector strip in pre-positioned blister packs (from coil)
  • Weld blister pack with cover film

System Parameters

  • Cycle time per coil: ≤ 0.2 seconds
  • Fully linked transfer line with “reel-to-reel” carrier tape processing
  • Wire diameter 32-50μm
  • Impulse test station
  • Aligning and bending of terminals on rotary indexing table

System Parameters

  • Cycle time per motor: 15 seconds
  • Motor housing is lifted and aligned using two centering pins
  • ECU unit is already centered and is gripped in a floating manner
  • The motor sleeve is heated and expanded using an inductor
  • The motor must be placed over the ECU unit within three seconds
