Semi-Automatic Assembly Systems

We design semi-automatic assembly lines according to customer requirements, utilizing various processes where the loading and feeding of components is performed manually. These systems can later be fully automated by equipping them with automatic material feeding systems.

System Parameters

Cycle time: 140 seconds per element

Technical availability: 95%

Number of stations: 7

Dimensions: approx. 18 m x 6 m

Very high cleanliness requirements

  • Manually moved fixtures for components
  • Laser marking system
  • Various linear and radial feeding systems, e.g.:
    • O-Ring ring feeders
    • Vanes
  • Joining processes with force/distance control
  • Various vision systems for position and presence inspection

Construction of semi-automatic assembly systems for hydraulic vane pumps with the highest precision and cleanliness requirements. Vane pumps are intended, for example, for use in high-speed gearboxes.

System Parameters

Cycle time: 10 seconds per element

  • WT Bosch circulation system
  • Feeding of various components using robots from multi-level rod storage
  • Various linear and radial feeding systems, e.g.:
    • Spring feeders
    • Ball feeders
  • Various pressing systems (with force/distance control)
  • Various vision systems
  • Functional and leak tests
  • Laser marking system

Expansion of the assembly systems for oil pumps with the integration of gearboxes of the highest precision, using a self-learning robot for assembling gear pairs.
